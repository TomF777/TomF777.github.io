Game "Fire the Ball"

This is simple game designed by using JavaScript and Canvas

The goal of the game is to click as many times as possible on the moving ball

As soon as you manage to click on the ball, the fire icon will appear and point score will be increased

Game difficulty is determined by ball motion speed which can be chosen with dropdown.
